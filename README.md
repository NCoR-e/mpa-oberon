## Oberon by HVR Group
### От идеи формирования макета с 0 до реализации (из MPA в SPA) (через параллельное обучение участников группы разработки)

### Данный проект представляет собой продукт интеллектуальной собственности (HVR Group) через параллельное обучение для членов группы HVR, поэтому он может быть дополнительно усовершенствован, внедрены новые технологии, а также проведена масштабная рефакторизация и замена используемых технологий по мере возникновения проблем или идей по улучшению.
